# FRIDAY — Cheatsheet

Just talk naturally. FRIDAY figures out which module to use.
Force a specific one anytime with the slash command.

---

## BUILD `/build`
Builds anything that runs in a browser. Saves to ~/Documents/FRIDAY/output/

**Say:**
- "build a landing page for a barber shop in Miami"
- "make a snake game, classic, use arrow keys"
- "create a digital invitation for my karting event"
- "build a pricing calculator for my agency"
- "make a restaurant menu page"
- "fix the bug in this code: [paste code]"

---

## OUTREACH `/outreach`
Finds leads + writes and sends cold emails via Brevo.

**Say:**
- "find restaurants in Miami that need a website"
- "find 15 barbers in New York"
- "write a cold email for a dentist in LA"
- "send to john@business.com for a gym that needs a site"
- "write a follow-up email for a prospect who didn't reply"

---

## CONTENT `/content`
Writes social media content + validates ads before you post them.

**Say:**
- "write 3 Instagram posts for my web design agency"
- "write a TikTok hook for a website launch"
- "make a content calendar for this week"
- "is this ad good to post? [paste copy]"
- "score this ad" + send a photo
- "rewrite this caption to convert better"

---

## SEO `/seo`
Audits websites and improves search rankings.

**Say:**
- "audit fileflowco.com"
- "what keywords should a web design agency in Miami target?"
- "write a meta title and description for my homepage"
- "why is my site not ranking on Google?"
- "what are my competitors doing better than me?"

---

## SUPPORT `/support`
Drafts replies to client messages and handles complaints.

**Say:**
- "client said: the website looks outdated, what do I say?"
- "client is asking how much a website costs"
- "client is upset about the timeline, help me reply"
- "write an onboarding message for a new client"
- "client wants a refund, help me handle this"

---

## PROJECTS `/projects`
Tracks active projects, deadlines, and what's pending.

**Say:**
- "add project: Nike site, deadline Friday, budget $1500"
- "what's pending?"
- "mark the barber shop site as complete"
- "what's overdue?"
- "create a project plan for a restaurant site"

---

## Tips
- Send a **photo** → automatically scores it as an ad
- Ask **anything else** → FRIDAY answers directly (no need for a command)
- Type **status** → see API usage and costs
- All built files are saved to: `~/Documents/FRIDAY/output/`
- All leads are saved to: `~/Documents/FRIDAY/leads/`
